# Pokédex App
Use the advanced list to explore Pokémon by type, ability, and more.

## Technologies
+ UIKit
+ Alamofire
+ MVP
+ Grand Central Dispatch
+ UserDefaults
+ Unit-testing
+ Swift Package Manager
